# Flat Desktop Wiki

A completely standalone, browser-based, self-updating Wiki designed to live directly on your desktop. No servers, no databases, just pure HTML/CSS/JavaScript.

## 🚀 Overview

The **Flat Desktop Wiki** brings the power and organization of a Wikipedia-style knowledge base directly to your local machine. It allows you to create, edit, and link pages dynamically within a single, portable environment.

### Key Features
- **Zero-Setup Local App:** Runs directly in your browser without needing Node.js, Apache, or any backend server.
- **Self-Updating:** Edits made in the browser are managed within the local file structure.
- **Easy Backups:** Because it's "flat," backing up your entire knowledge base is as simple as copying the folder to a safe location.
- **Classic Aesthetic:** Designed with a clean, readable, Wikipedia-inspired font stack.

## 🛠️ Installation & Usage

1. **Clone or Download:** 
   ```bash
   git clone https://github.com/elcafe7/flat_desktop_wiki.git
   ```
   *Or simply download the repository as a ZIP.*
2. **Open in Browser:** Double-click the `index.html` file to open it in any modern web browser.
3. **Start Editing:** Use the built-in interface to create new pages, interlink them, and build your personal knowledge base.
